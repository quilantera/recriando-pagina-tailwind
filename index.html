Get started with Tailwind CSS

Bem-vindo à documentação do Tailwind CSS! Aqui você encontrará todas as informações necessárias para começar a usar e explorar os recursos incríveis deste framework de estilo.

O Tailwind CSS é uma ferramenta poderosa e flexível que permite criar interfaces de usuário ricas e responsivas de forma eficiente. Com uma abordagem utility-first, o Tailwind CSS fornece uma ampla variedade de classes CSS prontas para uso, permitindo que você estilize seus elementos HTML de maneira rápida e consistente.

Explore a documentação completa do Tailwind CSS para aprender mais sobre cada um desses passos e obter exemplos práticos. Esteja preparado para se surpreender com a rapidez e eficiência que o Tailwind CSS traz ao seu fluxo de trabalho de desenvolvimento front-end.

Instalação

Para começar a usar o Tailwind CSS, você precisa instalá-lo em seu projeto. Existem várias formas de fazer isso, mas a maneira mais comum é por meio de um gerenciador de pacotes, como o npm ou o yarn. Veja abaixo um exemplo básico de instalação usando o npm:

1- Instale TailwindCSS

        
          npm install -D tailwindCSS
          npx tailwindCSS init

2- Configure seu template de patch

           
            /** @type {import('tailwindCSS').Config} */
            module.exports = {
              content: ["./src/**/*.{html,js}"],
              theme: {
                extend: {},
              },
              plugins: [],
            }

Estilos

Todos os botões foram estilizados de maneira padrão, tais como 

          
            button{
              display: block;
              margin: 2.75rem;
              min-width: 10rem;
              min-height: 2.75rem; 
              transition: 0.5s;
              font-size: 1.1rem;
              font-family: 'Inter', sans-serif; /*Necessário instalação*/
              text-transform: uppercase;
              font-weight: 500;
              padding: 0.75rem; 
              letter-spacing: 0.2px;
            }

Exemplo Botão 1

Neste exemplo, irei mostrar a estilização do botão do exemplo 1.

código HTML
                
                  ...
                  <button class="ex1"> Exemplo 1 </button>
                  ...
código CSS
                
                  .ex1{
                    border: 2px solid #E2E8F0;
                    background: #0B1121;
                    color: #E2E8F0;
                   
                  }
                  .ex1:hover{
                    background: #E2E8F0;
                    color: #0B1121;
                    box-shadow: 1px 1px 20px #E2E8F0;
                    letter-spacing: 0.6px;
                  }
